Base de los enlaces: El elemento vacío contiene una URL (mediante un atributo href) que sirve de referencia para el resto de URLs enlazadas en el resto del documento. Este elemento no se suele utilizar apenas. Enlaces: Los elementos vacíos enlazan a otros documentos para diferentes fines. El uso más habitual es para enlazar a hojas de estilo, aunque puede tener muchos usos. Pueden contener cualquier tipo de información, pero el agente que recibe la página web debe ser capaz de interpretar esa información. Algunos elementos son interpretados correctamente por todos los agentes, por ejemplo el que indica el juego de caracteres de la página web: Otra etiqueta cuyo uso se ha extendido en los últimos años es la que indica a los sistemas de pantallas de alta densidad que deben usar como ancho del documento el ancho de pantalla del dispositivo.
